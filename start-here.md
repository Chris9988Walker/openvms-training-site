# Start Here

Welcome to the OpenVMS System Engineering Training Suite.

This program is designed to build practical, real‑world skills for system operators, engineers, and administrators working with OpenVMS environments.

---

## What You Will Learn

Across eight modules, you will gain hands‑on experience with:

- File and directory operations  
- System monitoring and diagnostics  
- Disk and volume management  
- User and security administration  
- DCL scripting and automation  
- Networking and cluster operations  
- RMS file structures  
- System startup, shutdown, and configuration  

Each module includes:

- **Exam questions** to test your knowledge  
- **Hands‑on exercises** to build confidence  
- **Troubleshooting scenarios** to simulate real‑world issues  

---

## How to Use This Training

1. **Start with Module 1** and work through modules in order.  
2. Complete the **exam questions** before moving on.  
3. Perform the **hands‑on exercises** on a test OpenVMS system.  
4. Attempt the **scenario challenges** to build operational intuition.  
5. Instructors may provide access to the answer keys separately.  

---

## Recommended Prerequisites

- Basic familiarity with command‑line environments  
- Access to an OpenVMS test system (physical or virtual)  
- A willingness to experiment and explore
